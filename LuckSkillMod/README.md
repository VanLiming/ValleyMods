# 星露谷物语 - 幸运技能MOD

一个为星露谷物语添加**幸运技能**的MOD。

## 功能特性

- **新增幸运技能**：可以升级到10级
- **提升掉落率**：提高找到金品质物品的概率
- **增加金币收益**：卖出物品时获得更多金币
- **提升特殊事件触发率**：增加遇到幸运事件的概率
- **技能进度保存**：游戏自动保存你的幸运技能等级

## 安装方法

### 前置要求
1. 已安装 [SMAPI](https://smapi.io/) (Stardew Modding API)
2. 星露谷物语已更新至最新版本

### 安装步骤
1. 下载本MOD的最新版本
2. 将MOD文件夹解压到 `Stardew Valley/Mods` 目录下
3. 启动游戏，SMAPI会自动加载该MOD
4. 游戏中使用命令 `luck_info` 查看幸运技能信息

## 使用方法

### 游戏内命令

```
luck_info          - 查看当前幸运技能等级和经验值
luck_add_exp <数值> - 增加幸运技能经验值 (仅调试用)
luck_set_level <等级> - 设置幸运技能等级 (仅调试用)
```

### 获得幸运技能经验值

- **钓鱼**：每成功钓鱼获得2点经验
- **挖矿**：每敲碎岩石获得1点经验
- **农业**：每收获作物获得1点经验
- **寻宝**：捡到金品质物品获得5点经验
- **技能升级**：提升其他技能时获得1点经验

## 幸运技能效果

| 技能等级 | 效果 |
|--------|------|
| 1-2级 | 金品质物品概率 +5% |
| 3-4级 | 金品质物品概率 +10% |
| 5-6级 | 金品质物品概率 +15%、物品价格 +5% |
| 7-8级 | 金品质物品概率 +20%、物品价格 +10% |
| 9-10级 | 金品质物品概率 +25%、物品价格 +15% |

## 配置文件

生成的 `config.json` 可自定义各项参数：

```json
{
  "LuckBonusPerLevel": 2.5,
  "PriceMultiplier": 0.05,
  "QualityChanceIncrease": 5,
  "ExpPerActivity": {
    "Fishing": 2,
    "Mining": 1,
    "Farming": 1,
    "TreasureFind": 5
  }
}
```

## 常见问题

**Q: MOD会影响多人游戏吗？**
A: 这个MOD目前仅适用于单人游戏。多人模式支持计划在后续版本中加入。

**Q: 幸运技能会保存吗？**
A: 是的，幸运技能数据会随游戏存档自动保存。

**Q: 可以重置技能吗？**
A: 可以，在游戏中使用命令 `luck_set_level 0` 重置为0级。
