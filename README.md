# Valley Mods Collection

星露谷物语MOD合集 - 为游戏添加新的功能和特性。

## MOD列表

### 🍀 幸运技能 (Luck Skill MOD)

一个全新的技能系统，让你在星露谷物语中获得幸运加成。

**主要功能**：
- 新增幸运技能，可升级到10级
- 增加金品质物品掉落概率
- 提升物品售价
- 获得幸运技能经验的多种方式

**安装**：
1. 需要安装 [SMAPI](https://smapi.io/)
2. 将 `LuckSkillMod` 文件夹复制到游戏的 `Mods` 目录
3. 使用SMAPI启动游戏

**使用**：
- 游戏内输入 `luck_info` 查看技能信息
- 详见 `LuckSkillMod/README.md`

## 系统要求

- 星露谷物语 (1.6.0 或更新版本)
- SMAPI 3.13.0 或更新版本
- .NET 5.0 (用于编译)
